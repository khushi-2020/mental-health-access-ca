# Geographic Disparities in Mental Health Care Access — California Counties

A county-level analysis of mental health care **availability** and **affordability** across California, using the Two Dimensions of Access dataset (OSF, 2024–2025).

## Key Findings

- Rural counties charge the **highest median session fees ($167.50)** despite having the lowest disposable incomes — creating a 4.9% ten-session burden vs. 3.8% in urban areas
- Urban Medicaid acceptance is **1.3%** — functionally zero — while micropolitan counties accept Medicaid at 10× that rate
- **Sierra County** carries the highest burden: nearly 10% of disposable income for ten sessions
- Access deserts cluster in the **northern interior and Central Valley**

## View the Analysis

Open `index.html` in any browser to view the full interactive analysis.

## Data

This analysis uses the **Two Dimensions of Access** dataset (v1.0, January 2026), which draws on Psychology Today provider listings across U.S. counties. The dataset was created and made publicly available by its authors on OSF.

**Download the data:** [https://osf.io/6ezrx](https://osf.io/6ezrx)

The CSV is **not included in this repository**. Place it in the `data/` folder before running the R script:

```
data/
└── Two Dimension of access.csv
```

## Reproducing the Analysis

### Requirements

R (≥ 4.2). Install required packages once:

```r
install.packages(c(
  "tidyverse", "scales", "ggrepel", "tigris",
  "sf", "viridis", "patchwork", "gt", "here"
))
```

### Run

Open `R/analysis.R` in RStudio and run the script. Figures export to the `figures/` folder.
