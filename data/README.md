# Data

The dataset used in this analysis is **not stored in this repository**.

## Download Instructions

1. Go to [https://osf.io/6ezrx](https://osf.io/6ezrx)
2. Download the file: `Two Dimension of access.csv`
3. Place it in this `data/` folder

The analysis script expects the file at:
```
data/Two Dimension of access.csv
```

## Dataset Citation

Two Dimensions of Access dataset, v1.0, January 2026.
